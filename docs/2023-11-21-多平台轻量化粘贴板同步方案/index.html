多平台轻量化粘贴板同步方案

尝试过很多跨平台粘贴板同步方案，甚至动了自己写的念头，好在找到了比较完美的方案： KDE Connect 。

你能想到的想不到的平台都支持，而且功能很多，不只是剪贴板同步，还有虚拟触摸板、 通知同步、控制音乐/幻灯片、查看/回复信息、浏览器集成、远程访问设备文件……

跨局域网

功能很多，但缺点也很明显，KDE Connect 是为局域网设计，好在跨局域网互通也很简单。 可以用 Tailscale 、ZeroTier 之类的异地组网方案。

Android 10+ 后台同步剪贴板

从 Android 10 开始，出于安全考虑，禁止了 APP 后台读取剪贴板。要绕过这一限制有两种方案。

Magisk / Xposed

如果设备有 ROOT 此方案更优，只需勾选目标 APP 即可。

  • Magisk：Kr328/Riru-ClipboardWhitelist
  • Xposed：GamerGirlandCo/xposed-clipboard-whitelist

ADB

国产设备 ROOT 都很麻烦，好在 KDE Connect 提供了另一种方案，执行下面三条 adb 命令即可。

1
2
3

adb -d shell pm grant org.kde.kdeconnect_tp android.permission.READ_LOGS;
adb -d shell appops set org.kde.kdeconnect_tp SYSTEM_ALERT_WINDOW allow;
adb -d shell am force-stop org.kde.kdeconnect_tp;

Linux 上轻量化方案

KDE Connect 对 Windows、Android、macOS 系统都提供了一键安装包，iOS 也可以从 App Store] 下载，都比较方便。

Linux 虽然各发行版官方仓库都有收录，但因其依赖了很多 KDE 组件，对于非 KDE 用户来说依赖数量实在太多了，就算想静态编译也非常麻烦，Flatpak 体积又过大。

发现隔壁 GNOME 用 JavaScript 重写了 KDE Connect : GSConnect 。当然，它作为 GNOME 扩展自然也依赖了一些 GNOME 的组件，但如果只需要粘贴板同步功能， 可以精简掉多余依赖，将依赖数量降低到 1 —— gjs 。

# Maintainer: Kitty Panics

#pkgbase=
pkgname=gsconnect
pkgver=56
pkgrel=2
#epoch=1
pkgdesc="KDE Connect implementation for GNOME."
arch=('x86_64')
url="https://github.com/GSConnect/gnome-shell-extension-gsconnect"
license=('GPL2' 'MPL2')
depends=('gjs')
makedepends=('meson')
optdepends=('gnome-shell: Get all the functions'
            'evolution-data-server: Contacts Integration'
            'python-nautilus: File Manager Integration (Nautilus)'
            'gsound: Sound Effects'
            'libcanberra: Sound Effects'
            'ydotool: Generic command-line automation tool (Wayland)'
            'wl-clipboard: Command-line copy/paste utilities (Wayland)')
provides=("${pkgname}" "gnome-shell-extension-${pkgname}")
conflicts=("${pkgname}" "gnome-shell-extension-${pkgname}")
#backup=("")
#options=('')
#install=xyz.install
source=("${url}/archive/refs/tags/v${pkgver}.tar.gz"
        "${pkgname}-daemon.sh"
        "${pkgname}-preferences.sh")
#noextract=("")
sha256sums=('SKIP'
            'SKIP'
            'SKIP')

build() {
    cd "${srcdir}"/gnome-shell-extension-"${pkgname}"-"${pkgver}"
    mkdir build && cd build
    meson --prefix=/usr \
          -Dinstalled_tests=false \
          -Dfirewalld=false \
          -Dwebextension=false ..
    meson compile
}

package() {
    # mkdir
    install -dm755 "${pkgdir}"/usr

    # pkg files
    cd "${srcdir}"/gnome-shell-extension-"${pkgname}"-"${pkgver}"/build
    DESTDIR="${pkgdir}"/ meson install
    # bin
    install -Dm755 "${srcdir}"/gsconnect-daemon.sh \
                   "${pkgdir}"/usr/bin/gsconnect-daemon
    install -Dm755 "${srcdir}"/gsconnect-preferences.sh \
                   "${pkgdir}"/usr/bin/gsconnect-pref
}
