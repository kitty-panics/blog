批量解析顶级域名和二级域名

Kitty • Panics  included in Python3
 2018-12-11  235 words   One minute 
Contents

为方便描述，本文将 .com、.cn、.com.cn、.edu.cn 这类一级 / 二级域名统一定义为 顶级域名，对其下一级子域名定义为 二级域名。

最近有需求要将一批网址提取出其顶级域名和二级域名，示例如下。

得益于 Python 无所不包的包，如果不算空行、注释和导包的话，只需 3 行代码即可完成。

# -*- coding: UTF-8 -*-

# 导入包
import sys
from tld import get_tld

# 逐行处理输出
with open(sys.argv[1], 'r') as READ_DOMAIN:
    for GET_DOMAIN_LIST in READ_DOMAIN.readlines():
        print(get_tld(GET_DOMAIN_LIST, as_object=True).fld)
