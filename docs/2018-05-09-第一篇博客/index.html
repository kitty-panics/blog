第一篇博客

种一棵树，

最好的时间是十年前，

其次是现在。

大学伊始，就开始了写博客之路，直到今天，差不多快两年啦。

按照互联网行业的两年来算，两年能改变的东西，真的太多，当年的 “中华酷联” 好不风光， 转眼之间就变成了 “华米OV”。

文章去哪了？

虽然写了快两年的博客，但如你所见，截止现在，就只有这寥寥几篇的文章，那么

文章都去哪儿了？

这两年，换过好几个写博客的地方。

  • CSDN
    • 最开始使用的博客平台。不过这网站后来质量下降的太厉害，动不动删文，遂弃用
  • 博客园
    • 挺坚持初心的一个网站。可惜在一次重装系统时丢失了账号密码
  • 简书
    • 卿本佳人，奈何 从贼 ，直接把这网站加入黑名单
  • 购买服务器搭建
    • 由于忘记续费，服务器被回收

在这些网站、平台写的博客算是都丢失了吧。当时应该会觉得有些可惜吧，现在已经觉得无所谓， 毕竟很多博客都是转载的，原创没多少。当然吃一堑，长一智。为了密码不再丢失、不再忘记， 用上了密码管理器 (KeePassXC 一生推)。

要写个啥？

之前买的几个域名还没到期，想了想还是不能放着让它吃灰。所以，现在博客重新开张啦。

严格的来说，这并不是一个纯粹的技术博客 (其实我也很想纯粹，奈何技术不够)。

所以，就用来记录一下所思所想吧。

以前我并不是一个念旧的人，

或者说是讨厌念旧，

所以并没有从过去带来什么回忆。

可是人总有寂寞的时候，

终会有回想过去的时候。

在某个时刻，

发现自己想要回忆时，

却只能在那一点模糊的记忆中努力寻找，

似乎寻到，但伸手却又不可得。

才明白，

回忆之所以动人，

是因为在回忆中还可以重新选择，

可以将那些毫无关联的往事重新组合起来，

从而获得了全新的过去。

所以，还是记录一下吧，

或幼稚，或无聊，要不然日后凭何怀缅？
