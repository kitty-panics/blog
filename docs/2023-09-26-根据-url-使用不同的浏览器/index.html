作为 Firefox 用户，有些网站会恶意降低 Firefox 的体验，很是遭糕。从外部 URL 打开浏览器会调用系统默认浏览器，但如果不设置默认浏览器，又只能每次手动选择。

要解决这问题只能写个脚本，替换掉默认浏览器，再在这个前置脚本中判断使用哪款浏览器， 脚本大致逻辑如下。

# -*- coding: utf-8 -*-

import sys, os

url = sys.argv[1]
chrome_path = '"C:\\Program Files\\Chrome\\chrome.exe"'
firefox_path = '"C:\\Program Files\\Firefox\\firefox.exe"'

chrome_url_list = ['youtube.com', 'gmail.com']
firefox_url_list = ['baidu.com', 'weibo.com']

if url in chrome_url_list:
    os.system(f'start "" {chrome_path} {url}')
elif url in firefox_url_list:
    os.system(f'start "" {frefox_path} {url}')
# 回退到默认浏览器
else:
    os.system(f'start "" {chrome_path} {url}')

替换默认浏览器

新注册一个浏览器比较复杂，所以替换默认浏览器的启动程序更简单方便，就两步。

1. 先确定默认浏览器注册表名称

在 “命令提示符 (cmd)” 中执行下面的命令获取浏览器注册表名称。

1
2

reg query HKCU\Software\Microsoft\Windows\Shell\Associations\UrlAssociations\http\UserChoice /v ProgId
reg query HKCU\Software\Microsoft\Windows\Shell\Associations\UrlAssociations\https\UserChoice /v ProgId

比如 Chrome 浏览器的注册表名称为 ChromeHTML。

2. 再更改默认浏览器的启动程序

需 以管理员身份运行 打开 “命令提示符 (cmd)"，并执行下面的命令。

1
2
3
4

:: 格式
reg ADD HKLM\SOFTWARE\Classes\<浏览器注册表名称>\shell\open\command /f /d "\"<上述Python脚本路径>\" \"%1\""
:: 例子
reg ADD HKLM\SOFTWARE\Classes\ChromeHTML\shell\open\command /f /d "\"<上述Python脚本路径>\" \"%1\""
