禁用 Linux 系统的蜂鸣声

Kitty • Panics  included in Linux
 2018-08-22  223 words   One minute 
Contents
  • 临时禁用
  • 永久禁用

有时在终端或 TTY 中按下 Backspace、Tab 键会发出 “滴” 声，还是最大音量， 特别是在大半夜，会吓死人的。

搜索了一番，发现有很多解决方法，整理一下，做个记录。

临时禁用

有三种方法可以临时禁用。

  1. 移除 pcspkr 模块
    • sudo modprobe -r pcspkr
  2. 使用 setterm 命令 (由 util-linux 包提供)
    • setterm -blength 0
  3. 使用 xset 命令 (由 xorg-xset 包提供)
    • xset b off

永久禁用

有两种方法可以临时禁用。

  1. 移除 pcspkr 模块
    • 使用 /etc/modprobe.d
      • 在 /etc/modprobe.d 目录中创建内容为 blacklist pcspkr 的 nobeep.conf 文件
    • 添加内核参数
      • 在 Boot Loader 中添加 modprobe.blacklist=pcspkr 内核参数
  2. 配置 readline
    • 在 /etc/inputrc 配置文件中取消 set bell-style none 行首的注释
