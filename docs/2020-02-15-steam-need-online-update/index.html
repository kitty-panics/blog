Steam need online update

Kitty • Panics  included in 游戏
 2020-02-15  359 words   One minute 
Contents
  • 原因分析

打算把卸载掉的 Steam 装回来，结果运行安装程序时弹出 “Steam need online update” 错误提示。 给出的错误信息实在太少了，不过对于这种情况可以在终端运行试试，一般来说能获取到更多信息。

根据终端中的输出信息来看是访问一个 URL 失败了，把这 URL 复制出来用浏览器打开发现能正常打开， 是一个 JSON 文件，里面记录了很多文件的名字、大小、哈希值。

看来这是在下载完整安装包时失败了。即然如此，就手动下载吧。

# 进入 Steam 离线包下载目录
cd $HOME/.local/share/Steam/package

# 下载 JSON 文件
curl -LO "http://media.steampowered.com/client/steam_client_ubuntu12"

# 解析 JSON 文件并批量下载
cat steam_client_ubuntu12 | rg '"file"' | awk '{print $2}' | sed 's/"//g' | while read i; do curl -LO "http://media.steampowered.com/client/$i"; done
cat steam_client_ubuntu12 | rg '"zipvz"' | awk '{print $2}' | sed 's/"//g' | while read i; do curl -LO "http://media.steampowered.com/client/$i"; done

命令执行完后，再次运行 Steam 就一切正常了。

原因分析

Steam 安装程序访问 URL 失败，但浏览器和 curl 正常的原因应该是因为 Steam 不走系统的 DNS 和代理，然后 DNS 被劫持篡改了 (连接已重置？)。

而浏览器和 curl 都使用了 DNS over HTTPS ，所以是正常的。
