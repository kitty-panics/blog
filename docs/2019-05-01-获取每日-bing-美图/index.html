意外发现 Bing 主页图片其实有提供 API，这样的话就能用更优雅的方法解析了。

1

https://bing.com/HPImageArchive.aspx?format=js&idx=0&n=1

简要的 API 说明如下。

URL 组成部分 参数 含义
https://bing.com/HPImageArchive.aspx ? 无 无
format=js js (也就是 json), xml 输出图片信息所用的数据格式
&idx=0 0 (默认), 1, 2, 3, 4, 5, 6 ,7 输出 x 天前 (0 表示今天) 的图片信息
&n=1 1 (默认), 2, 3, 4, 5, 6, 7, 8 输出 x 天内 (1 表示今天) 所有的图片信息

 

 

Bing 的主页每天都会更新一张图片，拿来作壁纸不要太爽。

手动下载壁纸有一段时间，主要是有时候壁纸太丑，不想要，不过手动下载又实在麻烦的很。那就写个脚本呗。

# -*- coding:utf-8 -*-

#
# 下载 Bing 每日图片作为壁纸
#

# 导包
import os
import re
import sys
import json
import requests

# 定义图片 "存放目录"
PATH_WALLPAPER = ".local/share/Wallpaper/"
# 定义图片 "基本 URL"
URL_BASE = "https://bing.com"
# 定义图片 "API 地址"
URL_API = "https://bing.com/HPImageArchive.aspx?format=js&idx=0&n=1"

# 判断壁纸存放目录是否存在，如果不存在就创建
def DIR_Detection():
    isExists = os.path.exists(PATH_WALLPAPER)
    if not isExists:
        os.makedirs(PATH_WALLPAPER)


# 设置壁纸
def Set_Wallpaper(pic_name, pic_type):
    # 通过 feh 程序设置壁纸
    # 如果你想使用其它壁纸设置工具的话，请更改 APP_WALLPAPER
    APP_WALLPAPER = "feh --randomize --no-fehbg --bg-scale "
    os.system(APP_WALLPAPER + PATH_WALLPAPER + pic_name + pic_type)


# 下载壁纸
def DL_Wallpaper():
    # 访问 Bing API
    get_json = json.loads(requests.get(URL_API).text)
    # 解析图片的 URL
    parse_url = get_json['images'][0]['url']
    # 解析图片的 名称
    parse_name = re.search(r'OHR\.(.*)_ZH-CN', parse_url).group(1)
    # 解析图片的 格式
    parse_type = re.search(r'1920x1080(.*)\&rf=', parse_url).group(1)

    # 下载图片
    DL_JPG = requests.get(URL_BASE + parse_url)
    with open(PATH_WALLPAPER + parse_name + parse_type, 'wb') as f:
        f.write(DL_JPG.content)
        f.close
    # 设置壁纸
    Set_Wallpaper(parse_name, parse_type)


# 入口
DIR_Detection()
DL_Wallpaper()
