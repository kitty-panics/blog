2022-02-16 的更新

去官网瞥了一眼，没想到小马激活官网又提供激活工具了 (KMS10.zip)，当然鉴于下述原因，不要使用。

 

 

需要用回 Windows 7 系统，激活工具用的是 Oem7F7 (盗版行为请勿模仿)。下载 Oem7F7 时再次感受到了中文互联网那惊心动魄的下载体验。各种流氓软件满天飞，下载按钮各种藏雷。

这也是我坚持用 Linux 系统的原因 (所以就把原版 Oem7F7 删了)，当然本次用回 Windows 7 也是作业需要，迫不得已。

名称

小马激活工具 (也叫 Oem7) 最后一个版本 (Final7，也叫 Oem7F7) 发布在 2010 年 07 月 29 日， 之后就再也没有更新了。而且只能激活 Windows Vista、Windows 7 和 Windows Server 2008。

下载地址

小马激活工具其实有两个版本，即 Oem7F7 和 Oem7F7B。Oem7F7 是常规版，一般来说无脑用这版就行。 Oem7F7B 是特殊机型备用版，极少数机型 (例如三星 R428) 得用备用版才能进行激活。

  • Oem7F7
    • http://www.pccppc.com/download/oem7F.rar (已失效)
    • https://web.archive.org/web/20101105051913/http://www.pccppc.com/download/oem7F.rar
  • Oem7F7B
    • http://www.pccppc.com/download/oem7FB.rar (已失效)
    • https://web.archive.org/web/20101105051913/http://www.pccppc.com/download/oem7FB.rar

小马激活病毒

Oem7F7 停更后不久官网就移除了下载页面，现在网络上各种修改版 (病毒) 满天飞，还都说自己是原版。 详见 《“小马激活” 病毒新变种分析报告》 。

在这份报告中明确指出 原 “小马激活工具” 的制作团队可能与该病毒运作团队之间存在着直接关系， 没想到屠龙少年终成恶龙。所以不建议使用 Oem7F7 之后的任何版本，就算它声称自己是官方。

至于 Oem7F7，最后一版停留在 2010 年，而第一代病毒出现在 2015 年。我用过很多病毒扫描工具扫描过， 也对其行为监控了一段时间，最后一版 Oem7F7 应该是没有问题的，可以放心食用。

哈希值

如果你需要下载最后一版 Oem7F7，又不知道是不是下载到了病毒，可以对比一下文件的哈希值。
