i3wm 中工作区含指定程序时关闭护眼模式

Kitty • Panics  included in Linux
 2019-06-05  713 words   2 minutes 
Contents
  • 改造 i3wm
  • 编写外部脚本 eye

作为长时间使用电脑的人群，电脑上启用护眼模式真的很重要。 但是护眼模式会使屏幕色温发生改变，颜色也就不正常了，而像追剧、 修图、游戏等场景又需要得到正确的颜色反馈。

每次都是手动开关护眼模式，时间一久就很是麻烦了，所以捣鼓一下在 i3wm 中实现工作区含指定程序时关闭护眼模式。

改造 i3wm

i3wm 内建只支持简单的窗口检测和操作，其配置文件也只能按照指定格式编写， 不能在里面实现复杂的逻辑。所以需要编写一个外部脚本来实现。

修改 i3wm 的配置文件 ~/.config/i3/config，让其每次改变 Workspace 时执行一次外部脚本 eye。

# Define names for default workspaces for which we configure key bindings later on.
# We use variables to avoid repeating the names in multiple places.
set $ws1 "1"
set $ws2 "2"
set $ws3 "3"
set $ws4 "4"
set $ws5 "5"
set $ws6 "6"
set $ws7 "7"
set $ws8 "8"
set $ws9 "9"
set $ws10 "10"

# switch to workspace
bindsym $mod+1 workspace $ws1;  exec --no-startup-id eye
bindsym $mod+2 workspace $ws2;  exec --no-startup-id eye
bindsym $mod+3 workspace $ws3;  exec --no-startup-id eye
bindsym $mod+4 workspace $ws4;  exec --no-startup-id eye

bindsym $mod+9 workspace $ws9;  exec --no-startup-id eye
bindsym $mod+0 workspace $ws10; exec --no-startup-id eye

编写外部脚本 eye

在外部脚本 eye 中，使用 xdotool 命令获取当前焦点窗口名， 当检测到有指定的窗口时就关闭护眼模式，反之启用。使用 Shell 实现，完整代码如下。

#!/usr/bin/env bash

#
# 当前 Workspace 包含指定窗口时关闭护眼模式，反之启用
#

# 定义需要关闭护眼模式的窗口名 (一行一个)
PROGRAM_NAME="Steam
GNU 图像处理程序
Krita
爱奇艺 - Google Chrome
腾讯视频 - 中国领先的在线视频媒体平台
高清1080P在线观看平台 - Google Chrome"

# 获取当前活动窗口名
WINDOW_NAME=$(xdotool getwindowfocus getwindowname)

# 检测其中是否包含指定窗口程序
for i in "$PROGRAM_NAME"; do
    DETECT=$(echo -e "$WINDOW_NAME" | grep "$i")
    if [[ "$DETECT" != '' ]]; then
        # 包含。关闭护眼模式
        killall redshift
    else
        # 不包含。检测是否需要启用护眼模式 (已启用无动作，未启用则启用)
        ps -ef | grep "redshift" | grep -v grep
        if [[ "$?" == "0" ]]; then
            :
        else
            killall redshift
            exec redshift &
        fi
    fi
done
