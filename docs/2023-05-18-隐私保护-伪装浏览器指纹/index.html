隐私保护 —— 伪装浏览器指纹

虽说现在越来越多的网络服务只提供 APP，还有些就算提供网页版也需要登录， 但还是有隐私保护的习惯。

IP 地址

IP 地址应该算是最好伪装的，得益于国内分配到的 IPv4 太少，现在运营商都在用 NAT ， 重启光猫就可以得到一个新 IP (手机可以开关一次飞行模式)。

但如果用代理时想要伪装真实 IP，还需要防范通过 WebRTC 泄漏真实 IP，泄漏原理就是 WebRTC 允许点到点通信，中间的代理服务器就失效了。

可以使用 WebRTC Control 插件保护真实 IP。

UserAgent

UserAgent 里面一般会包含浏览器及其版本，操作系统，设备型号。

可以使用 Random User-Agent (Switcher) 插件生成随机 UserAgent。

AudioContext 指纹

可以使用 AudioContext Fingerprint Defender 插件生成随机噪音来指纹。

Canvas 指纹

可以使用 Canvas Fingerprint Defender 插件生成随机噪音来指纹。

Font 指纹

可以使用 Font Fingerprint Defender 插件生成随机噪音来指纹。

WebGL 指纹

可以使用 WebGL Fingerprint Defender 插件生成随机噪音来指纹。

WebGPU 指纹

可以使用 WebGPU Fingerprint Defender 插件生成随机噪音来指纹。

语言、时区

这两个内容虽然也可以用浏览器插件伪装，但直接定义环境变量也可以。

export TZ="Asia/Singapore"
export LANG="en_US.UTF-8"
export LANGUAGE="en_US:en:C"

exec /usr/bin/google-chrome

如果要每次启动随机化环境变量，可以通过随机读取 /etc/locale.gen 和 /usr/share/zoneinfo 里的内容来实现。

远不于此

当然隐私保护远不于此，需要长期查漏补缺，比如还可以：

  • 安装 Google Analytics Opt-out Add-on (by Google) 插件禁用 Google 分析。
  • 禁用第三方 Cookie 防止被跨网站跟踪。
  • 安装 Cookie AutoDelete 插件自动清理不需要保持状态的网站的 Cookie。
  • 安装 LocalCDN 、Decentraleyes 防止被 CDN 跟踪。
  • 使用 Tor 隐藏出口 IP。
  • ……
