常用汉字中平均每字的字母长度

多年音形码、形码用户，当初抛弃全拼就是因为码长太长，重码太多。打了很久的单字， 感觉有时码长并不比全拼短，但个人主观感受是不准确的，遂统计一下。

将 常用汉字 定义为 《通用规范汉字表》 ，其拼音数据采用 Unihan 收录的拼音。

《通用规范汉字表》

使用 Python 来统计，先说结论：3.3343615052436766。

# -*- coding: utf-8 -*-

import re

with open('Unihan/Unihan_Readings.txt', 'r') as file:
    py_len, char_num = 0, 0
    for i in file.readlines():
        if re.search('^U+.*\tkTGHZ2013', i):
            char_num += 1
            for j in re.findall(r':(\w+)', i.strip().split('\t')[2]):
                py_len += len(j)
    print(f'汉字个数为：{char_num}')
    print(f'拼音长度为：{py_len}')
    print(f'平均长度为：{py_len / char_num}')

# -*- coding: utf-8 -*-

import re

with open('Unihan/Unihan_Readings.txt', 'r') as file:
    py_len, char_num = 0, 0
    for i in file.readlines():
        if re.search('^U+.*\tkHanyuPinyin', i):
            char_num += 1
            for j in i.strip().split(':')[1].split()[0].split(','):
                py_len += len(j)
    print(f'汉字个数为：{char_num}')
    print(f'拼音长度为：{py_len}')
    print(f'平均长度为：{py_len / char_num}')

结果为: 4.014268971579256。
