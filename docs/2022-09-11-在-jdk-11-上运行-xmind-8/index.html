这个错误是由于某处代码尝试使用 Java 反射机制来访问 java.net.URLClassLoader.addURL() 方法， 但由于 Java 9 开始引入模块化系统，某些类和方法默认情况下不再对所有模块开放。

Fix

要解决也很简单，使用 --add-opens 选项即可。

1

java --add-opens=java.base/java.net=ALL-UNNAMED 主类

而对于 XMind 8 只需在主配置文件 (即 /usr/share/java/xmind8/XMind.ini) 中加入此参数即可。

1

--add-opens=java.base/java.net=ALL-UNNAMED
